# サイト所有権確認コード管理ワークフロー

外部サービス（Impact.com, Google Search Console, Pinterest等）から要求されるサイト所有権確認コードの配置箇所を管理します。

## 配置場所の原則
1. **メタタグ（推奨）**: `app/layout.tsx` の `metadata.verification` オブジェクトに追加します。
2. **HTMLタグ（要求時）**: `app/page.tsx` （ホームページ）の `main` セクションの先頭に非表示要素として配置します。

## 現在の配置状況
### Impact.com (Affiliate)
- **コード**: `Impact-Site-Verification: 204b2504-8f49-41e6-b979-1a1077b19bdb`
- **場所**: `app/page.tsx` の `<main>` タグ直下
- **配置日**: 2026-04-07

## 手順
1. 要求されたコードをコピーする。
2. 上記「配置場所の原則」に従い、適切なファイルに追記する。
3. ブラウザ（または `curl`）でソースコードを表示し、コードが含まれていることを確認する。
4. 本ワークフローの「現在の配置状況」を更新する。
