# Shop Verification Guide

## 方針

ショップ詳細の「日本発送」「関税」「送料」は、推定ではなく根拠ベースで表示する。

表示ルール:

- `確認済み`
  - 説明文がある
  - 根拠URLがある
- `要再確認`
  - 説明文はある
  - 根拠URLがない
- `未確認`
  - 説明文がない

## 必要な項目

ショップごとに、できるだけ次を持つ。

- `shipping_guide`
- `tax_guide`
- `fee_guide`
- `shipping_url`
- `tax_url`
- `fee_url`

## 更新ルール

1. 公式の配送ポリシーやヘルプページを確認する
2. 根拠URLを保存する
3. 要点だけを `*_guide` にまとめる
4. 更新後は `updated_at` が変わる状態で保存する

## 表示上の考え方

- 根拠URLがないものは「確認済み」とは表示しない
- 未確認の項目は、購入前に公式サイト確認を促す
- 推定で「注文しやすい可能性」などの曖昧な断言は避ける
